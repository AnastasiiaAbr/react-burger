
## Учебный проект Яндекс Практикума

Проект выполнен в рамках обучения на платформе Яндекс Практикум. Основной акцент сделан на отработке подходов к разработке клиентских приложений, работе с состоянием, типизации и тестированию.

**Используемые технологии**

***React*** – разработка интерфейса

***Redux Toolkit*** – управление состоянием

***TypeScript*** – статическая типизация

***WebSocket*** – обмен данными в реальном времени

***Jest, Cypress*** - тестирование
