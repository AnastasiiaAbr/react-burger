**STELLAR BURGER** - приложение для сборки бургеров в режиме реального времени.

Пользователь может формировать заказ из списка ингредиентов, создавать заказ и отслеживать его статус.


  

**Основные возможности:**

  

 - Просмотр ингредиентов по категориям;

  
  

 - Конструктор бургера с перетаскиванием элементов (Drag and Drop);

  
   

 - Авторизация и регистрация, восстановление пароля;

   
  

 - Создание заказа и просмотр истории заказов в реальном времени;

   

 - Защищённые роуты;

  

**Tехнологии:**

 -  React;
   
     
   
  - React Router;
   
     
   
   - Redux Toolkit;
   
     
   
   - React DnD;
   
     
   
   - TypeScript;
   
     
   
   - WebSocket;
   
     
   
   - Jest / Cypress для тестирования.
